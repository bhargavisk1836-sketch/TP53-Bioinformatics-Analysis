# TP53-Bioinformatics-Analysis
End-to-end bioinformatics analysis of the human TP53 gene using NCBI, BLAST, EMBOSS Needle, Clustal Omega and UniProt.

## About
TP53 is one of the most important tumor suppressor genes in humans.
This project demonstrates how different bioinformatics tools can be integrated to study gene sequence, protein function, sequence conservation and evolutionary relationships.
The project was completed as the capstone project during my Bioinformatics Summer Internship.

## Objectives
- Retrieve TP53 sequence from GenBank
- Perform BLAST analysis
- Compare homologous sequences
- Perform Pairwise Alignment
- Perform Multiple Sequence Alignment
- Build Guide Tree
- Build Phylogenetic Tree
- Analyze protein information using UniProt
- Interpret evolutionary relationships

  ## WorkFlow
NCBI -> Sequence Retrieval -> BLAST -> Pairwise Alignment -> Multiple Sequence Alignment -> Guide Tree -> Phylogenetic Tree -> UniProt -> Biological Interpretation

## Key Findings
- TP53 is highly conserved across primates.
- Chimpanzee showed the highest similarity to the human TP53 sequence.
- Conserved regions correspond to functional domains responsible for DNA repair and apoptosis.
- BLAST, EMBOSS Needle and phylogenetic analysis produced consistent evolutionary relationships.
